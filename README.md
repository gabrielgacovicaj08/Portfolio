# Gabriel Gacovicaj — Portfolio

A personal developer portfolio built with React, TypeScript, and Vite. It has a terminal/hacker aesthetic with dark themes and monospace fonts throughout.

## Sections

- **Hero** — intro with a typewriter effect
- **About** — personal info displayed as a terminal file output
- **Projects** — live GitHub repos fetched from the GitHub API, filtered to exclude forks
- **Contact** — links to GitHub, LinkedIn, and email

## Tech Stack

- React + TypeScript
- Vite
- Framer Motion — scroll-triggered fade/slide animations
- GSAP + ScrollTrigger — staggered animations for cards and contact rows

## Running locally

```bash
npm install
npm run dev
```
